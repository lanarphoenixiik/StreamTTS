# 🎙️ StreamTTS Portable — умная озвучка чата Twitch

**Готовое портативное приложение** для стримеров, которое читает сообщения из чата голосом (TTS) с русским акцентом, не нагружая компьютер. Работает из одной папки — можно переносить на флешке.

## 📌 Что умеет программа

| Функция | Описание |
|---------|----------|
| 🗣️ **Озвучка сообщений** | Читает вслух сообщения из чата Twitch с помощью локального синтезатора речи **Piper TTS** (русский голос). |
| ⏳ **Умная очередь** | Сообщения не читаются мгновенно, а ставятся в очередь. Это даёт задержку 5–15 секунд при флуде — вы успеваете отреагировать, и процессор не перегружается. |
| 🔇 **Режим тишины (можно выключить)** | Анализирует звук с вашего микрофона. Если вы говорите — TTS **ждёт паузы** (вы не перебиваете друг друга). В настройках можно полностью отключить этот режим. |
| 🤖 **Игнор ботов** | Не читает сообщения от известных ботов (Nightbot, StreamElements, Moobot и др.) — список настраивается. |
| 🧹 **Фильтрация сообщений** | Убирает ссылки, команды (!команда), упоминания (@ник), эмодзи и слишком длинные тексты (обрезает). |
| ⌨️ **Горячая клавиша** | `Ctrl+Shift+M` — временно отключить/включить озвучку (мьют). Работает, когда окно консоли активно. |
| ⚙️ **Веб-интерфейс настроек** | Удобная панель в браузере (http://127.0.0.1:5000) для изменения всех параметров без редактирования файлов. |
| 🔒 **Скрытие токена** | Поле ввода токена скрывает символы, рядом есть кнопка-глазик для показа/скрытия. |
| 💾 **Сохранение настроек** | Все настройки хранятся в `config.json` в корневой папке. Можно редактировать вручную или через веб-интерфейс. |
| 📦 **Портативность** | Всё приложение в одной папке. Не требует установки в систему. Работает на любом Windows-компьютере. |

## 🖥️ Системные требования

- Windows 7 / 8 / 10 / 11 (64-bit)
- Микрофон (для режима тишины) — опционально
- Подключение к интернету (для подключения к Twitch)
- ~200 МБ свободного места

## 🚀 Быстрый старт

##  1. 📦 Установка

1. Клонируйте репозиторий или скачайте архив.
2. Запустите `setup.bat` — он автоматически скачает и установит:
   - Портативный Python 3.10
   - Piper TTS
   - Русскую голосовую модель
   - Все необходимые Python-библиотеки
3. Отредактируйте `config.json`, указав свои данные (токен, канал).
4. Запустите `start.bat`.

> **Примечание:** Установка может занять несколько минут из-за скачивания компонентов.

### 2. Получите OAuth-токен Twitch

- Перейдите на [twitchtokengenerator.com](https://twitchtokengenerator.com/)
- Нажмите **Generate Token** → выберите разрешения `chat:read` и `chat:edit`.
- Скопируйте полученный токен (начинается с `oauth:`).

### 3. Отредактируйте конфиг

Откройте файл `config.json` в корневой папке и укажите:

```json
{
  "twitch_token": "oauth:ваш_токен",
  "twitch_nick": "имя_вашего_бота",
  "twitch_channel": "имя_канала_стримера",
  "piper_model": "voices/ru_RU-...-medium.onnx"
}

Ctrl + Shift + E     Экстренная остановка — прерывает текущее озвучивание и очищает очередь.
