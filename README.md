## ✨ Особенности

| Функция | Описание |
|---------|----------|
| 🗣️ **Озвучка чата** | Читает сообщения из чата Twitch голосом на русском языке |
| 🎭 **Разные голоса для ролей** | Отдельные голоса для модераторов, подписчиков, VIP, стримера и обычных зрителей |
| 🎚️ **Выбор голосов через веб-интерфейс** | Легко меняйте модели из выпадающих списков |
| 🤫 **Режим тишины** | Программа ждёт, пока вы замолкнете, чтобы не перекрывать ваш голос |
| 🛑 **Экстренная остановка** | Горячая клавиша `Ctrl+Shift+E` — мгновенно прерывает воспроизведение и очищает очередь |
| 🌐 **Веб-интерфейс настроек** | Все настройки доступны через браузер (http://127.0.0.1:5000) |
| 📦 **Портативность** | Не требует установки — работает из одной папки |
| ⚡ **Автоустановка** | `setup.bat` скачивает и настраивает всё автоматически |
| 🧹 **Фильтрация сообщений** | Убирает ссылки, команды, упоминания и эмодзи |
| ⏳ **Умная очередь** | Задержка 5–15 секунд при флуде — не перегружает процессор |

## 🖥️ Системные требования

- **Windows 7 / 8 / 10 / 11 (64-bit)**
- **Микрофон** (для режима тишины, опционально)
- **Подключение к интернету** (только для первоначальной установки и подключения к Twitch)
- **Около 200 МБ** свободного места

## 🚀 Быстрый старт

### 1️⃣ Клонируйте или скачайте репозиторий

```bash
git clone https://github.com/lanarphoenixiik/StreamTTS.git
```

или скачайте ZIP и распакуйте.

### 2️⃣ Запустите установку
Дважды кликните по setup.bat. Он автоматически:

Загрузит портативный Python 3.10

Установит pip и необходимые библиотеки

Скачает и настроит Piper TTS

Загрузит 4 русские голосовые модели: Ирина, Денис, Дмитрий, Руслан

⏳ Время установки: ~5–10 минут (зависит от интернета).

### 3️⃣ Настройте токен и канал
Откройте файл config.json в корневой папке. Замените:

"twitch_token": "oauth:ваш_токен",
"twitch_nick": "имя_бота",
"twitch_channel": "имя_канала"

Токен получите на twitchtokengenerator.com, выбрав права chat:read и chat:edit.

### 4️⃣ Запустите приложение
Дважды кликните по start.bat. Откроется консоль и автоматически браузер с веб-интерфейсом.

Если браузер не открылся — перейдите вручную: http://127.0.0.1:5000

Увидите сообщение в консоли:

text
✅ Бот ... подключен к каналу ...
Теперь сообщения из чата будут озвучиваться!

## 🎛️ Веб-интерфейс
В браузере по адресу http://127.0.0.1:5000 доступны настройки:

Twitch — токен, ник бота, канал

Piper TTS — путь к piper.exe, голос по умолчанию, скорость речи

Голоса для ролей — выберите модель для модератора, подписчика, VIP и стримера из выпадающего списка

Микрофон — включение/отключение режима тишины, порог чувствительности, задержка

Фильтры — список игнорируемых ботов, максимальная длина сообщения, размер очереди

После изменения настроек нажмите «Сохранить и перезагрузить» — изменения применятся без перезапуска.

## 🎭 Голосовые модели
В комплекте идут 4 модели. Вы можете назначить их для разных ролей:

Имя в интерфейсе	Файл	Описание
Ирина (женский)	ru_RU-irina-medium.onnx	Голос по умолчанию
Денис (мужской)	ru_RU-denis-medium.onnx	Назначается модераторам
Дмитрий (мужской)	ru_RU-dmitri-medium.onnx	Назначается стримеру
Руслан (мужской)	ru_RU-ruslan-medium.onnx	Назначается VIP
Вы можете легко изменить эти назначения через веб-интерфейс.

## ⌨️ Горячие клавиши

---
|Комбинация|Действие|
|---------|----------|
|Ctrl + Shift + M|Включить/выключить озвучку (мьют)|
|Ctrl + Shift + E|🛑 Экстренная остановка — прерывает текущее воспроизведение, очищает очередь, удаляет временный файл|
|--------------------|
|Важно: Горячие клавиши работают, только когда окно консоли находится в фокусе.|

## ⚙️ Конфигурация
Все настройки хранятся в config.json в корневой папке. Вот полный список параметров:

Параметр	Описание	По умолчанию
twitch_token	OAuth-токен для чата	"oauth:..."
twitch_nick	Имя бота (необязательно)	"ваш_бот"
twitch_channel	Имя канала (без #)	"ваш_канал"
piper_exe	Путь к piper.exe	"piper/piper.exe"
default_model	Голос по умолчанию	"voices/ru_RU-irina-medium.onnx"
role_models	Голоса для ролей	(указаны выше)
mic_device_id	ID микрофона (null=авто)	null
rms_threshold	Чувствительность микрофона	0.015
silence_delay	Задержка перед озвучкой (сек)	0.5
silence_enabled	Включить режим тишины	true
speech_speed	Скорость речи (0.5–2.0)	0.8
ignored_users	Игнорируемые боты	["streamelements", ...]
max_queue_size	Максимальный размер очереди	15
message_max_len	Максимальная длина текста	150
mute_hotkey	Горячая клавиша для мьюта	"ctrl+shift+m"

🧠 Устранение проблем

---
|Проблема|Решение|
|---------|----------|
|❌ Ошибка аутентификации Twitch|Токен истёк или не имеет прав. Сгенерируйте новый на twitchtokengenerator.com с правами chat:read и chat:edit.|
|❌ Piper завершается с ошибкой (код 3221226505)|Несовместимость версий. Удалите папку piper, перезапустите setup.bat для обновления Piper.|
|❌ Нет звука|Проверьте пути к моделям, выберите правильное устройство вывода в Windows. Если Bluetooth-гарнитура — отключите режим «Hands-Free».|
|❌ Бот не видит сообщения|Проверьте регистр имени канала, пишите с другого аккаунта (бот игнорирует свои сообщения).|
|❌ Веб-интерфейс не открывается|Проверьте, не занят ли порт 5000. Откройте вручную http://127.0.0.1:5000.|
|❌ Режим тишины не работает|Укажите ID микрофона вручную через веб-интерфейс. Отрегулируйте порог rms_threshold.|
